# 历史邮件样本

把导出的历史邮件放在这个文件夹里。

## 建议命名

```text
001-aigc.txt
002-social.txt
003-game.txt
004-cdn.txt
```

## 脱敏方式

- 公司名保留城市、行业和公司格式，关键名字换成 `xx`：`广州xx网络科技有限公司`
- 项目编号保留前几位数字，后面换成 `xxx`：`20260417xxxxxxx`
- 账号 ID 保留前几位，后面换成 `x`：`2479xxxxxx`
- 联系人、手机号、邮箱、微信、真实人名直接替换成：`联系人A`、`手机号A`、`邮箱A`

## 尽量保留

- 产品名称
- 折扣
- 返佣
- 申请有效期
- 客户背景
- 项目背景
- 竞争对手情况
- 代理商贡献说明
- 审批结果或你后来发现的问题
