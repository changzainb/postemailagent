# 特批价格邮件生成器

给腾讯云渠道经理发送特批价格、折扣和返佣申请时使用。第一版是纯静态页面，不需要登录、不需要数据库，也不直接发送邮件。

## 怎么用

1. 打开 `index.html`。
2. 先填客户名称和客户行业/业务。
3. 点击“自动匹配产品”，工具会按经验规则带出产品。
4. 对每个产品选择默认折扣、突破折扣、原价，也可以直接手动改折扣。
5. 没匹配到或少产品时，继续手动新增产品。
6. 代理商信息和渠道经理称呼固定带出，申请有效期默认按当天自动生成。
7. 申请理由里填写成立年份、公司规模、目前项目情况，点击“生成理由”。
8. 调整产品、折扣和返佣，查看右侧阈值提醒。
9. 复制邮件标题和正文，粘贴到邮箱发送。

## 当前字段

- 渠道经理称呼
- 项目编号
- 代理商全称、腾讯云账号 ID
- 客户公司全称、腾讯云账号 ID
- 申请产品、折扣、返佣
- 客户行业/场景推荐产品
- 默认折扣、突破折扣、原价、手动折扣
- 折扣、返佣经验阈值提醒
- 申请有效期
- 成立年份、公司规模、目前项目情况
- 自动润色客户背景、项目背景
- 自动生成竞争对手情况
- 固定代理商贡献说明

默认规则：代理商全称、代理商腾讯云账号 ID、渠道经理称呼为固定配置；申请开始日期为打开页面当天，申请结束日期为明年当天的前一天。

竞争对手规则：普通客户从阿里云、AWS、华为云、百度云、金山云、京东云、七牛云、白山云里随机取 1-2 个；语音聊天社交客户优先从即构、声网里取。只选 1 家时写“给客户报过价”，选 2 家时写“都给客户报过价”。

## 怎么沉淀案例

每个案例尽量补这些信息，后面可以直接变成行业规则。也可以直接复制 `CASE_TEMPLATE.md` 的格式给我：

- 客户行业/业务类型
- 客户规模和项目背景
- 实际申请产品
- 最终审批通过的折扣
- 是否有返佣、返佣比例
- 哪些产品不能返佣
- 哪些折扣低于某个点会明显难批
